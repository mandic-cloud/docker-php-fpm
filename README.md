# docker-php-fpm
Dockerfiles for PHP Flavors


## Usage:

Install [docker](https://docs.docker.com/engine/installation/) and [docker-compose](https://docs.docker.com/compose/install/), review the [docker-compose.yml](./docker-compose.yml), and run:

```
docker-compose up
```
